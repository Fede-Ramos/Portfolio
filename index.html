Stack de tecnologías

Puedo definir a Javascript como el lenguaje principal a la hora de desarrollar mis proyectos, y con el que me siento más cómodo y familiarizado. Estoy siempre pendiente de sus actualizaciones y en continuo aprendizaje ya que es muy dinámico. También cuento con conocimientos en Python, lenguaje que me resultó muy amigable e interesante.

Dentro del stack tecnológico, en el backend he trabajado con arquitecturas API REST, con PostgreSQL como base de datos, Sequelize en caso de requerir un ORM y Node.js como intérprete de Javascript y así poder correr mis servidores. Express para realizar el ruteado y Postman o ThunderClient para realizar las peticiones.

En el frontend, adhiriendo a html y css, destaco a React.js tecnología que considero muy completa y efectiva, y he tenido la oportunidad de aplicarla tanto en proyectos pequeños como grandes, sumando Redux para manejo de estados y por supuesto alguna herramienta de css, como Material UI o Bootstrap.
